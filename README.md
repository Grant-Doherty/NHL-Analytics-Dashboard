

The goal of this project was to build a visual dashboard that displays current league wide data from the National Hockey League,
along with insightful metrics like momentum, rolling averages, L10 and more. Current NHL data can be viewed as time series charts, and 
metrics in the chart can be plotted to visualize season trends. 

'dashboard.py' drives the app, and pulls information and functions from 'dash_functions.py', 'load_2024.py', and 'load_2025.py'. 
The file 'Update 2025 Game Data.py' interacts with an API to request and store published data from the current NHL season.
